This repository contains a collection of data analytics exercises completed as part of a Modern Data Analytics course. The projects and assignments included demonstrate foundational and applied skills in the data analytics workflow, including data cleaning, exploratory data analysis, visualization, and basic modeling techniques.

Across the exercises, I worked with real-world datasets to practice identifying trends, summarizing key insights, and communicating results effectively. The work emphasizes reproducible analysis using tools such as R and/or Python, along with structured reporting and interpretation of findings.
